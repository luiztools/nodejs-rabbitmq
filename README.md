# nodejs-rabbitmq
A simple Node.js + RabbitMQ example.

## To Run
1. Run npm install
2. Run a RabbitMQ server
3. Run webapi.js
4. Run worker.js
5. Make some POST requests to http://localhost:3000 to see in action!
